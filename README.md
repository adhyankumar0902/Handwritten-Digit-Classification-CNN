# Handwritten Digit Classification using Perceptron, ANN, and CNN

## Project Overview

This project compares three machine learning/deep learning models for handwritten digit recognition using the MNIST dataset.

Models implemented:
- Perceptron
- Artificial Neural Network (ANN)
- Convolutional Neural Network (CNN)

The project demonstrates how model complexity affects classification performance.

## Dataset

- MNIST Handwritten Digits
- Training data: `mnist_train.csv`
- Testing data: `mnist_test.csv`

Each image is 28 × 28 grayscale pixels and belongs to one of 10 digit classes (0–9).

## Technologies Used

- Python
- TensorFlow / Keras
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Workflow

1. Load dataset
2. Data preprocessing
3. Train Perceptron
4. Train ANN
5. Train CNN
6. Compare model performance
7. Evaluate accuracy
